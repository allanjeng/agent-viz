# agent-viz

Real-time pixel-art visualization dashboard for OpenClaw agent infrastructure.

## MVP Structure

- `bridge/` Node.js WebSocket bridge service with mocked OpenClaw state sources
- `frontend/` React + PixiJS v8 pixel office dashboard
- root workspace scripts to run both together

## Quick Start

```bash
npm install
npm run dev
```

Local ports:
- frontend: `http://localhost:3000`
- bridge WebSocket: `ws://localhost:3001`

## Scripts

- `npm run dev` run bridge + frontend together
- `npm run dev:bridge` run bridge only
- `npm run dev:frontend` run frontend only
- `npm run build` build both workspaces
- `npm run import:vendor` import/normalize Donarg + Cozy paid tilesets into frontend runtime atlas

## Vendor Asset Setup (Donarg + Cozy)

1. Put paid source files in:
- `assets/vendor/donarg/`
- `assets/vendor/cozy-furniture/`

2. Expected source filenames:
- `assets/vendor/donarg/Donarg-Office-Tileset.zip` or `assets/vendor/donarg/donarg-office-tileset.png`
- `assets/vendor/cozy-furniture/Cozy-Furniture.zip` or `assets/vendor/cozy-furniture/cozy-furniture-tileset.png`

3. Run importer:

```bash
npm run import:vendor
```

4. Generated runtime files:
- `frontend/public/assets/vendor/furniture.png`
- `frontend/public/assets/vendor/furniture.json`

If vendor assets are not present, the scene automatically falls back to built-in procedural desks/chairs/props.

For detailed drop-zone instructions, see `assets/vendor/README.md`.

## Bridge Data Contract

Bridge pushes WebSocket messages in this form:

```ts
{
  type: "state",
  data: {
    agents: Array<{
      id: string;
      name: string;
      type: "persistent" | "contractor";
      status: "idle" | "thinking" | "working" | "error";
      connections: string[];
    }>;
    stats: {
      sessionCount: number;
      activeContractors: number;
      updatedAt: string;
    };
    events: Array<{ id: string; timestamp: string; message: string }>;
  }
}
```

## Notes

- OpenClaw gateway/session integrations are mocked for MVP but source classes are isolated for easy swap-in.
- The bridge still watches `~/.acpx/sessions/*.json` and emits mock file-change events.
- Pixel assets are generated by `frontend/scripts/generate_assets.py` into `frontend/public/assets/`.
