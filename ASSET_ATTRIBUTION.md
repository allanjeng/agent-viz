# Asset Attribution

This project uses only free prebuilt asset sources for the office scene.

## 1) OpenGameArt Pixel Art Lab / Office Tiles
- Source links:
  - <https://opengameart.org/sites/default/files/bgtiles_1.png>
  - <https://opengameart.org/sites/default/files/bgtiles_2.png>
- Local source files:
  - `assets/free-sources/opengameart/lab-office/bgtiles_1.png`
  - `assets/free-sources/opengameart/lab-office/bgtiles_2.png`
- License note: free to use; author requests attribution/credit on use.
- Usage in repo: extracted into `frontend/public/assets/free/office.png` via `scripts/import_free_assets.py`.
